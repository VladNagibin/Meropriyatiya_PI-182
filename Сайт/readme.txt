Запуск сервера у себя:
1)Скачать и установить node js https://nodejs.org/en/
2)Скачать Visual Code https://code.visualstudio.com/
3)Запускаем Visual Code -> Open folder -> Папка гитхаба/Сайт
4)Сверху на панели Terminal -> new terminal
5)В терминале пишем npm run start и нажимаем enter
6)Заходим на https://2ip.ru/ и скидываем мне свой айпи чтобы я дал доступ к бд
7)Открываем браузер и в строке URL пишем localhost:3000
